# NorthFile 0.1.1

这是 `0.1.0` 之后的一次界面与交互整理版本，重点完成了侧边栏、文件树、“我的电脑”主页，以及窗口标题栏主题联动的收口。

## 主要更新

- 重构侧边栏普通模式与 compact 模式
- 侧边栏支持固定、文件树、云盘、网络、标签分组
- compact 模式下支持图标入口和右侧弹出菜单
- “我的电脑”主页支持驱动器列表展示
- 驱动器页显示名称、类型、总大小、可用空间
- 文件树支持更稳定的展开、收起、路径跳转与选中同步
- 修复无权限目录导航后右侧列表状态异常的问题
- 优化状态栏、sidebar 底部设置区与主界面整体布局
- 标题栏现在可跟随应用浅色/深色模式，保持与窗口材质一致

## 交互改进

- 区分文件树箭头展开和节点点击跳转
- 收起上层节点后，重新展开可恢复之前的选中层级
- 路径变化时，左侧树会优先选中当前可见的上层节点
- compact 模式下，“我的电脑”支持多级目录弹出菜单导航
- 固定分组、分组标题、普通项 hover 与选中样式进一步统一

## 发布说明

- 当前发布目标为 `win-x64`
- 发布包已包含 .NET 运行时与 `rust_engine.dll`
- 当前版本仍属于早期可用版本，后续会继续完善性能、搜索、侧边栏体验与发布流程

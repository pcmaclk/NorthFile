# NorthFile 0.1.0

NorthFile `0.1.0` 是项目的首个可运行版本，当前重点完成了文件浏览主链路、导航体验和基础发布形态的整理。

## 本版本内容

- 文件列表默认按“文件夹优先、名称升序”排序
- 面包屑导航支持长路径折叠与溢出菜单
- 侧边栏支持拖拽调节宽度，并提供压缩模式
- 文件列表支持分页加载
- 文件大小、修改时间等详情信息支持异步补齐
- 支持重命名、删除、双击打开
- 补充了基础文件类型图标与应用图标
- 发布形态调整为可直接启动的 unpackaged `x64` 桌面程序

## 技术方向

- 界面层采用 WinUI 3
- 文件系统访问与目录处理由 Rust 后端负责
- 排序在后端完成，分页基于排序后的稳定结果集推进

## 当前限制

- 当前主要按 `x64` 目标进行验证
- 普通权限运行时，目录浏览通常会走传统枚举路径
- 原始 NTFS 访问仍属于增强路径，不作为默认运行前提
- 项目仍处于持续迭代阶段，后续还会继续优化大目录性能和搜索能力

## 运行说明

- 直接使用发布输出目录中的 `NorthFile.exe` 启动
- 当前版本不要求先安装 MSIX 包

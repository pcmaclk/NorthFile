# 设置功能实现进度

## 当前状态

设置页框架、导航、本地化和基础视觉样式已经完成。当前进入第一批真实功能接入阶段，优先实现低风险、可立即生效的设置项。

## 第一批已接入的功能

### 左侧 sidebar 分组显示

已接入以下设置项：

- 收藏
- 云
- 网络
- 标签

实现方式：

- 设置页 `通用` 分组中的开关改为可交互控件。
- 设置修改后立即同步到左侧 `SidebarView`。
- 设置值持久化到 `ApplicationData.Current.LocalSettings`。
- 应用重启后自动恢复上次设置。

涉及代码：

- `FileExplorerUI/Settings/AppSettings.cs`
- `FileExplorerUI/Settings/AppSettingsService.cs`
- `FileExplorerUI/Controls/SettingsView.xaml`
- `FileExplorerUI/Controls/SettingsView.xaml.cs`
- `FileExplorerUI/SidebarView.xaml.cs`
- `FileExplorerUI/MainWindow.xaml.cs`

# 已知问题

## 1. USN 仍为入口级能力
- 当前已完成能力探测与刷新策略接入。
- 尚未完成完整 USN Journal 增量消费（按 USN 游标读取并增量更新目录索引）。

## 2. NTFS 原始设备访问权限
- 普通权限下，部分机器对 `\\\\.\\X:` 打开会返回 Access Denied。
- 现有策略：自动 fallback 到 Traditional 枚举，功能可用但性能上限受影响。

## 3. 搜索仍为目录级扫描
- 当前在“当前目录”范围内筛选，不含深层递归索引。
- 超大目录下仍存在冷启动扫描成本。

## 4. 缓存一致性
- 目前采用 TTL + 修改时间 + 手动失效策略。
- 在极端频繁变更场景可能出现短暂显示延迟，后续由 USN 增量更新进一步收敛。

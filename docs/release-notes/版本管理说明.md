# 版本管理说明

## 单一版本源
- 根目录 `VERSION` 作为统一版本号来源（SemVer：`x.y.z`）。

## 受控文件
- Rust 引擎版本：`rust_engine/Cargo.toml` 的 `version`
- UI 版本：`FileExplorerUI/FileExplorerUI.csproj` 的
  - `Version`
  - `InformationalVersion`
  - `FileVersion`
  - `AssemblyVersion`
- 发布记录：`CHANGELOG.md`
- Release note 存档：`docs/release-notes/<version>.md`

## 升级版本
在项目根目录执行：

```powershell
pwsh .\scripts\bump-version.ps1 -Version 0.2.0
```

脚本会自动更新：
- `VERSION`
- `rust_engine/Cargo.toml`
- `FileExplorerUI/FileExplorerUI.csproj`

## 运行时检查
- Rust FFI 暴露 `fe_get_engine_version`
- UI 通过 `RustBatchInterop.GetEngineVersion()` 读取并显示在窗口标题

用于确认当前 UI 实际加载的 `rust_engine.dll` 版本，避免 DLL 不一致。

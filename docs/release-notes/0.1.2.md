# NorthFile 0.1.2

本版本是一次稳定性收口更新，重点完成收藏交互一致性、窗口尺寸记忆链路收敛，以及文件树路径跟随滚动体验修正。

## 主要更新

- 收藏区右键菜单接入主菜单链路，行为与普通文件夹菜单保持一致。
- 收藏项支持实时同步（外部改名/删除可自动反映到收藏展示）。
- 窗口尺寸记忆改为关闭窗口与最小化到托盘时直接采样真实窗口尺寸并保存。
- 启动恢复与首次激活应用链路补齐，窗口尺寸恢复更稳定。
- TreeView 路径跟随滚动调整为原生 `BringIntoView + VerticalOffset`，越界时自动留一行缓冲。
- 深色模式标题栏主题与窗口主题保持一致，避免 Mica 观感异常。

## 维护项

- 诊断日志降噪：保留关键节点日志，移除高频过程性输出，便于问题定位与长期运行。
